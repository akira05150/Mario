# Mario with java
將此專案資料夾放入Eclipse並編譯執行。<br>
此為一個網路上找的專案經過修改後，使用Java視窗程式設計出的簡易瑪莉歐程式，移動角色獲取coin分數並跳過陷阱，最後碰到終點結束遊戲。

## 新增功能
1. 修復人物控制的bug
2. 新增分數機制
3. 新增音效
4. 新增失敗與通關條件
